# Arcanum Testing

Test harness utilities for Arcanum apps. Provides `TestKernel` (a real
HyperKernel/RuneKernel composed for in-process dispatch) and `Factory` (a
reflection-based DTO generator that respects validation attributes).

> **Status:** package skeleton. The classes in this directory are placeholders
> while the testing-utilities arc lands incrementally. See `PLAN.md` for the
> active checklist.
